### workflow

- Run `hugo server -t hello-friend-ng`
- Run `hugo new posts/my-first-post.md`
- Edit the above file
- Check out [http://localhost:1313/](http://localhost:1313/)
- Repeat Edit
- Once done, commit and push
- Run `hugo`; It creates `public` folder!
